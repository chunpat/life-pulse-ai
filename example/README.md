# 掌控生活 首篇图卡使用说明

这个目录里现在有一份独立的本地合成页，可以把首篇 8 张推广图先快速排出来，不依赖当前项目启动状态。

## 文件说明

1. compose.html：首版 8 张 4:5 图卡预览页。
2. 01-home.png 到 09-analytics-week.png：图卡里用到的真实产品截图素材。

## 怎么打开

1. 在 Finder 里进入 example 目录，双击 compose.html。
2. 也可以把 compose.html 直接拖进浏览器打开。
3. 如果你更习惯本地服务，可以在这个目录下临时起一个静态服务再访问 compose.html，但不是必须。

## 怎么导出

### 方案一：逐张截图

1. 打开 compose.html。
2. 向下滚动，每张卡片都是独立 4:5 画布。
3. 用系统截图逐张截取即可。

这个方案最适合你快速改字、改顺序、看整体风格。

### 方案二：打印导出 PDF

1. 打开 compose.html。
2. 点击页面右上角的“打印 / 导出 PDF”。
3. 在浏览器打印预览里确认是一张卡片一页。
4. 选择“存储为 PDF”导出。

这个方案适合一次性导出 8 张图的标准版。

1. 图 8 的试用入口文案和二维码，现在是占位说明。

1. 先看图 1、图 2、图 8 的标题是否像你的语气。
2. 再看图 3、图 4、图 5 的截图是否需要换成更清晰的数据状态。
3. 最后替换真实体验地址，再统一导出。
